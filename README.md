# Hello Kitty & Friends — Card Tracker

Interactive checklist to track your Hello Kitty and Friends sticker collection (209 cards).

## Features

- Tap/click any card to mark it as collected
- Progress bar and counters update instantly
- Filter by **All**, **Collected**, or **Needed**
- Collapsible sections by theme
- Progress saved in your browser (localStorage) — works offline
- Mobile-friendly

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your tracker will be live at `https://<username>.github.io/<repo-name>/`
